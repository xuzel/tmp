## 十一、选项中常见的"干扰项词汇"（认识它们 = 秒排除）

这些词**几乎总是出现在错误选项里**：

| 干扰词 | 为什么是陷阱 |
|---|---|
| **efficiency** | 效率——不是伦理目标 |
| **speed** | 速度——不是伦理目标 |
| **cost** | 成本——商业指标非伦理目标 |
| **profit** | 利润——商业，非伦理 |
| **accuracy** | 准确率——技术指标 |
| **complexity** | 复杂度——技术指标 |
| **computation / computational cost** | 计算成本——技术指标 |
| **storage** | 存储——技术概念 |
| **visual appeal** | 视觉美观——UI 设计 |
| **model size** | 模型大小——技术指标 |
| **programming language** | 编程语言——技术工具 |
| **algorithm type** | 算法类型——技术分类 |
| **model architecture** | 模型架构——技术概念 |
| **data source** | 数据源——技术概念 |

⚠️ **规律**：看到"technical / commercial / performance 类词" → **几乎是错选项**；看到"ethical / responsible / equitable / transparent / human oversight" → **可能是答案**。
